## Git Commands

Use these if you want the CLI version:

```bash
git status

git push origin main
```

## GitHub Desktop Steps

1. Open GitHub Desktop.
2. Select the `personal-ai-trader` repository.
3. Review the changed files in the left panel.
4. Use a commit summary like:
   - `docs: add deployment and architecture documentation`
5. Add an optional description with:
   - `Cleaned repo-facing paths, added docs set, added Vercel prep, and created roadmap/update artifacts.`
6. Click `Commit to main`.
7. Click `Push origin`.

## Repository Settings Checklist

For a private repo:

- set repository visibility to `Private`
- add a short description
- add topics/tags
- add the production website later
- pin the repo on your profile only if you want it visible to approved viewers
